# python_4
python task week 5


This repository contains my Week 5 Object-Oriented Programming (OOP) tasks implemented in Python. The programs demonstrate core OOP concepts such as Inheritance, Polymorphism, Abstraction, and Method Overriding through real-world examples like a Smart Vehicle Management System, Online Payment System, School Notification System, Shape Drawing Application, Employee Salary System, and Online Learning Platform. These tasks helped me understand how to design classes, create inheritance hierarchies, override methods, and apply abstract classes to solve practical problems using OOP principles.
